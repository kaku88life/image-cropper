隱私權政策

圖片裁切工具 Chrome 擴充功能

本擴充功能不會收集、傳輸或儲存任何個人資料到外部伺服器。

資料收集

本擴充功能：

  • 不收集任何個人識別資訊
  • 不追蹤您的瀏覽行為
  • 不傳輸任何資料到外部伺服器
  • 不使用任何分析或追蹤工具

本地儲存

本擴充功能使用 Chrome 的本地儲存空間（chrome.storage.local）僅用於：

  • 暫時儲存您透過右鍵選單選取的圖片 URL，以便在擴充功能彈出視窗中載入

此資料僅儲存在您的本機裝置上，圖片載入後即會自動清除，不會傳送到任何外部服務。

圖片處理

所有圖片編輯和處理都在您的瀏覽器本地進行：

  • 圖片裁切、繪圖、濾鏡等操作完全在本地執行
  • 處理後的圖片直接下載到您的裝置或複製到剪貼簿
  • 不會將任何圖片上傳到雲端或外部伺服器

權限說明

本擴充功能請求以下權限：

  • activeTab：用於從當前頁面載入您選取的圖片
  • contextMenus：用於在圖片上顯示右鍵選單「使用圖片裁切工具編輯此圖片」
  • storage：用於暫時儲存選取的圖片 URL（僅在本地）

第三方服務

本擴充功能不使用任何第三方服務、API 或分析工具。

政策更新

如果本隱私權政策有任何更新，我們將在此頁面發布修訂版本。

聯絡方式

如果您對本隱私權政策有任何疑問，請透過 GitHub Issues 與我們聯繫。
