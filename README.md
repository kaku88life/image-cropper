# Image Cropper 圖片編輯工具

一個簡單易用的圖片裁切與編輯工具，提供 Chrome 擴充套件和網頁版兩種使用方式。

可以直接在網頁/瀏覽器上裁切、簡單修圖，將圖片裁切成指定尺寸後下載或直接複製。

## 功能特色

### 裁切功能
- 支援多種預設輸出尺寸（16:10、16:9、4:3、1:1、3:2、2:3、9:16）
- 自由裁切模式
- 自訂尺寸輸出
- 8 個調整端點，自由調整裁切範圍

### 繪圖工具
- **筆刷類型**：鉛筆、簽字筆、水彩筆、蠟筆、螢光筆
- **形狀工具**：矩形、圓形、箭頭、直線
- **特效工具**：馬賽克、模糊
- **其他**：文字輸入、填色桶
- 可調整顏色與筆刷粗細
- 支援復原 / 重做

### 輸出選項
- 支援 PNG、JPEG、WebP 格式
- 預覽處理結果後再下載
- 一鍵複製到剪貼簿

## 使用方式

### 載入圖片
- 點擊上傳區域選擇檔案
- 拖放圖片到上傳區域
- 使用 Ctrl+V 貼上剪貼簿中的圖片

### 網頁版
直接開啟 `web/index.html` 即可使用。

### Chrome 擴充套件安裝
1. 開啟 Chrome，進入 `chrome://extensions/`
2. 開啟右上角「開發人員模式」
3. 點擊「載入未封裝項目」
4. 選擇 `chrome-extension` 資料夾
5. 點擊工具列上的擴充套件圖示即可使用

## 專案結構

```
image-cropper/
├── chrome-extension/     # Chrome 擴充套件版本
│   ├── manifest.json
│   ├── popup.html
│   ├── popup.css
│   ├── popup.js
│   └── icons/
└── web/                  # 網頁版本
    ├── index.html
    ├── style.css
    └── script.js
```

## 技術特點

- 純前端實作，無需後端伺服器
- 使用 Canvas API 進行圖片處理
- 輸出畫質依原圖解析度，不因預覽縮小而降低品質
- 支援拖放、貼上等多種圖片載入方式
